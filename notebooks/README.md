# Notebooks

Use this folder for Jupyter notebooks (`.ipynb`) used during exploration,
visualization, checks, and one-off analysis.

Notebooks are useful for trying ideas quickly, inspecting intermediate data,
making figures, and documenting exploratory steps. When notebook code becomes
part of the regular workflow, move the reusable logic into `modules/` or a
script under `pipeline/`.

Suggested practice:

- Keep notebooks organized by user, topic, or experiment.
- Use clear filenames that describe the purpose of the notebook.
- Avoid committing large outputs, embedded data, or temporary results.
- Store generated figures, tables, reports, and other outputs in `results/`.
