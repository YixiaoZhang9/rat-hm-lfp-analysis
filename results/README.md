# Results

Use this folder for generated analysis outputs, figures, tables, summaries,
reports, and other result files.

This directory is intended for local outputs from notebooks and pipeline
scripts. It can be used to collect intermediate checks, final plots, validation
results, and written reports for sharing or review.

Suggested practice:

- Organize results by user, analysis type, date, or experiment.
- Include enough context in filenames so outputs are easy to trace later.
- Keep large raw data, processed data, and model files outside Git.
- If a result is important for collaboration, document how it was generated.
